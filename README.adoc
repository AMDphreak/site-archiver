= Site Downloader
:toc: macro

image:https://img.shields.io/github/license/AMDphreak/site-downloader[License]

A comprehensive toolset for archiving websites to a static, offline format.

== Components

* **site-downloader-lib**: D-lang library for crawling and link rewriting.
* **site-downloader-cli**: Command-line interface.
* **site-downloader-gui**: Webview-based GUI with page preview and session management.
* **site-downloader-extension**: Chrome extension (WASM-ready) for capturing sessions and triggering downloads.

== Quick Start

To start archiving a website immediately, ensure you have the link:https://dlang.org/download.html[D compiler and Dub] installed, then run:

[source,powershell]
----
cd apps/cli
dub run -- --url https://example.com
----

== Usage

=== CLI

[source,powershell]
----
cd apps/cli
dub run -- --url https://example.com --depth 2
----

=== GUI

[source,powershell]
----
cd apps/gui
dub run
----

=== Browser Extension

Load `apps/extension` as an unpacked extension in Chrome/Edge to trigger downloads directly from your browser.

TIP: For comprehensive usage instructions, see the link:docs/modules/ROOT/pages/index.adoc#_cli_arguments[CLI Arguments] documentation.

== Development & Building from Source

If you want to contribute to the project or build the documentation locally, follow these steps.

=== Prerequisites
* D Compiler (DMD or LDC)
* Dub
* Node.js & pnpm
* corepack enabled

=== Setup
[source,powershell]
----
corepack enable
pnpm install
----

The project uses a monorepo structure:
* `libs/archiver`: Core crawling and link rewriting logic.
* `apps/cli`: CLI frontend.
* `apps/gui`: GUI frontend using `webview-d`.
* `apps/extension`: Chrome/Edge extension.
* `docs`: Antora-based documentation.
